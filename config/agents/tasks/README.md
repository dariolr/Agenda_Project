# Tasks

Questa cartella contiene solo task correnti per agenti AI.

## Regole

- Un file per task attivo.
- Nome consigliato: `TASK_<nome>.md`.
- Non archiviare qui task storici lunghi.
- I task storici vanno eliminati se non servono più.
- Solo task storici ancora utili possono stare in `config/docs/archive/tasks/`.

## Uso

Ogni task deve indicare esplicitamente i file agent da leggere.
