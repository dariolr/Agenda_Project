# Agents Documentation

Questa cartella contiene solo istruzioni operative brevi per agenti AI/Codex.

## Come usare questi file

Per ogni task, leggere solo:

1. `config/agents/GLOBAL_RULES.md`
2. `config/agents/MONOREPO_MAP.md`
3. i file del progetto coinvolto
4. eventuali file feature in `config/agents/features/`
5. il task corrente in `config/agents/tasks/`

Non leggere tutto il monorepo se il task è circoscritto.

## Regola

- `config/agents/` = regole operative brevi
- `config/docs/` = documentazione canonica lunga
- `config/database/` = documentazione database
- `config/agents/tasks/` = task correnti per agenti AI
